## Recommandations techniques

### Format et qualité
- **Format** : JPG ou WebP (WebP recommandé pour de meilleures performances)
- **Qualité** : 80-85% pour JPG, 85-90% pour WebP
- **Taille maximale** : 200-300 KB par image
- **Optimisation** : Compresser les images avant upload (utiliser TinyPNG, Squoosh, ou ImageOptim)

### SEO et accessibilité
- Toutes les images ont déjà des balises `alt` optimisées dans le code
- Les balises alt contiennent des mots-clés pertinents pour le SEO
- Les images sont chargées en lazy loading (sauf l'image hero)

### Sources d'images recommandées
- **Unsplash** : https://unsplash.com (gratuit, haute qualité)
- **Pexels** : https://pexels.com (gratuit, haute qualité)
- **Pixabay** : https://pixabay.com (gratuit)
- **Canva** : https://canva.com (gratuit avec compte, création d'infographies)

### Mots-clés de recherche pour trouver les images
- "mathematics", "data science", "student studying", "education", "learning", "anxiety", "confidence", "career", "salary", "professional"

## Note importante

Si vous n'avez pas encore les images, le site fonctionnera quand même. Les images seront simplement manquantes. Vous pouvez :
1. Utiliser des images placeholder temporaires
2. Créer les images progressivement
3. Utiliser des services comme Unsplash pour trouver des images libres de droits

Les balises `alt` sont déjà en place pour le SEO, même si les images ne sont pas encore ajoutées.
